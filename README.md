# My Profile

A personal portfolio website built with HTML and CSS. Showcases my projects, skills, and certificates.

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Home page with bio, project cards, skills grid, and certificates |
| `GuitarPedal.html` | Project page for the Guitar Pedal with embedded YouTube demo |
| `WaveformDisplay.html` | Project page for the Waveform Display with embedded YouTube demo |

## Features

- Responsive layout that adapts to mobile and desktop
- Navigation bar with links to Home, LinkedIn, and GitHub
- Project cards linking to dedicated project pages with video demos
- Skills organised in a grid by category
- Certificate links to PDF documents

## Tech Stack

- HTML
- CSS (custom styling with gradient background, card layout, responsive grid)
